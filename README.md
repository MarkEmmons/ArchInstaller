#### To install

```shell
wget https://raw.github.com/MarkEmmons/ArchInstaller/master/install.sh -O - | bash
```

Make sure to delete all previous partitions. You can use gdisk or a one liner such as:

```shell
sgdisk --zap-all /dev/sda
```
